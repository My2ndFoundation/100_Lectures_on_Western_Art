## 一句话定义

诞生于 20 世纪初**德国**的第一个现代绘画流派——背后思想根源是德国对抗法国 [[实证主义 Positivism]] 的 [[新康德主义 Neo-Kantianism]]——把康德的自由意志用 **"民族"** 和 **"时代"** 两个维度加以**窄化**——所要表现的"绝不是个人的情感和情绪，而必然是关乎德意志民族的，是关乎时代的"（顾衡 069）。

## 时间范围

- 20 世纪初兴起于德国（顾衡 069）；常用断代约 **1905–1930s**（*not from wiki*）

## 所属时代

- 20 世纪上半叶

## 核心特征

### 思想根源（069 导论）

- **法德下半场的德国侧**：1860 年以后，[[孔德 Auguste Comte]] [[实证主义 Positivism]] 在法国大行其道并孵化出 [[现实主义 Realism]] / [[印象派 Impressionism]] / [[新印象主义 Neo-Impressionism]] / [[象征主义 Symbolism]] / [[立体主义 Cubism]]——1870 普法战争战胜方的德国不肯当法国小跟班——回到 [[伊曼努尔·康德 Immanuel Kant]] 那里另起炉灶 → [[新康德主义 Neo-Kantianism]]。
- **从康德的内部矛盾出发**：康德的 "普遍立法" 原则需要打三个补丁（**意志自由 / 上帝存在 / 灵魂不死**），自我否定了道德先验性。新康德主义从这个不能自洽处开始**扬弃**康德——**两条路径**：
  - [[黑格尔 G.W.F. Hegel]]：区分道德 / 伦理，否认道德先验，从**时代精神**找答案
  - [[费希特 Johann Gottlieb Fichte]]：不动道德先验根子，把共同体**窄化到民族**
- **"民族 + 时代"双维窄化**：两条路径都被新康德主义继承，成为表现主义最核心的思想支撑。
- **三流派定位（顾衡 069 一句话）**：
  > 浪漫主义是个人的，实证主义是普世的，而表现主义则是**民族的，和当下的**。

## 与其他流派的关系

- **承接 [[野兽派 Fauvism]]**：[[马蒂斯 Henri Matisse]] 的色彩解放与简化对德国表现主义产生重大影响（062 明示）
- **与 [[立体主义 Cubism]] 同处 20 世纪初前卫艺术格局**——但立体主义是法国实证主义路线产物，表现主义则是德国新康德主义路线产物，**两条路线相对垒**（069）
- **对照 [[浪漫主义 Romanticism]]**：浪漫主义是个人的——以"个体情感强度"对抗启蒙；表现主义则是民族的、当下的——以"共同体窄化"对抗实证（069）
- **对照法国现代诸流派**：[[现实主义 Realism]] / [[印象派 Impressionism]] / [[新印象主义 Neo-Impressionism]] / [[象征主义 Symbolism]] / [[立体主义 Cubism]] 都是 [[实证主义 Positivism]] 旗下的"普世真理"路线——表现主义则是新康德主义的"民族 + 时代"路线
- **历史背景**：[[神圣罗马帝国 Holy Roman Empire]] 的合同 / 规定训练造就了德国民族的"刺猬"性格（[[刺猬与狐狸 Hedgehog and the Fox]]）——是德国为什么会孵化出此种"集体性 + 时代性"现代主义的远因

## 076 综合——理论建构闭环 + Willing→Passion 切换

顾衡 076 把表现主义的**理论建构**与**语义切换**两块讲清楚：

### 理论建构的闭环

- **思想底层** = [[新康德主义 Neo-Kantianism]] 把康德道德先验性以**民族 + 时代**两维窄化（069 已建）
- **从康德到叔本华**：[[叔本华 Arthur Schopenhauer]] 完成"意志至上"体系化（《作为意志和表象的世界》）——世界本质是意志，现象是意志客体化
- **从生命意志到艺术意志**：[[李格尔 Alois Riegl]] / [[沃尔夫林 Heinrich Wölfflin]] 借用，提出 [[艺术意志 Kunstwollen]] —— 沃尔夫林"石头滚下山坡"比喻：山坡坑洼 = 外界互动，万有引力 = 艺术意志，本质是**纯粹的形式**
- **沃林格的临门一脚**（1911）：[[沃林格 Wilhelm Worringer]] 提出 [[南方写实-北方抽象 South-Realism vs North-Abstraction]] 二元论 → 明确**德意志民族艺术意志** → 定义为 [[抽象冲动 Urge to Abstraction]] → 具体表现 = [[向二维平面回归 Return to Flatness]] → **理论建构完成**

### Willing → Passion 的语义切换（**核心转折**）

- **原初含义**：表现主义所要表现的是叔本华的 **Willing（意志）**——既有不可抗拒的冷酷，又含**对个人哭哭啼啼情感的蔑视**——**与 Emotion / Passion 相反**。表现 Passion 那是 [[浪漫主义 Romanticism]]——这是个非常重要的区别。
- **切换的历史动因**：纳粹上台后，德语区艺术史学者跑到英国美国——**表现主义理论中的德国元素**不好用了，必须被**遮蔽和重构**。
- **切换的操作手法**：学者们抬高了 **[[凡·高 Vincent van Gogh]]**（"我的创作只是为了最充分地表现自己"）和 **[[诺尔德 Emil Nolde]]**（"我有意避开一切思考，因为艺术家要顺从他的本能去创作"）——通过这两位的抬高，表现主义从德语 Willing 切换到英语 Passion。
- **同化机制的德语回响**：早在德语区时代，[[立普斯 Theodor Lipps]] 就在 [[狄尔泰 Wilhelm Dilthey]] 为心理学开门的基础上提出 [[移情说 Theory of Empathy]]——主张观者可通过移情感受到画家内心 Emotion / Passion。
- **移情说的两路德语原生诘问**：
  - [[波普尔 Karl Popper]]：艺术家技术层面的算计让他变成"演员"，画面变成"舞台效果"——切断画家内心→画面通路
  - [[沃林格 Wilhelm Worringer]]：观者无法**原路返回**到艺术家那里——切断画面→观者通路
- **困境样本**——[[柯科施卡 Oskar Kokoschka]] [[风中新娘 The Bride of the Wind]] (1913–1914)：
  - **作者解读**：云雨后新娘心满意足睡着，新郎陷入深深的绝望，"心爱的女人正离他而去"
  - **模特（[[艾尔玛 Alma Mahler]]）解读**："你看看，他把自己画得像个国王"——强烈的控制欲
  - **顾衡 076 结论**：同一幅画连模特和画家解读都不一样。当表现主义表达**民族性**时**起码可被描述**；当表达**个人 Emotion / Passion** 时，**创作者与观者根本尿不到一个壶里去**——这正是表现主义脱离德国家乡后面临的**最大难题和困惑**。

### 一句话记忆

- 表现主义 = 德意志民族艺术意志 → 抽象冲动 → 向二维平面回归
- 表现主义所表现的 = **Willing**（叔本华的"生命意志"在画面上的投影），**不是** Emotion，更**不是** Passion
- 今天英语世界讲的"表现主义" = 切换语义后的产物，**和 100 年前德语区的表现主义完全不是一个东西**
