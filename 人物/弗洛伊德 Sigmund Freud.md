## 简介

奥地利精神病学家、神经科医生，[[精神分析 Psychoanalysis]] 学说创立人。1899 年发表《梦的解析》、1905 年发表《性学三论》，精神分析正式出炉面世。世纪之交在维也纳活动，**与 [[克里姆特 Gustav Klimt]] 是好朋友**——这一私交是 [[席勒 Egon Schiele]] 接触到精神分析的渠道（顾衡 074）。顾衡 074："弗洛伊德给德国人当时的仇女厌女症披了一层科学的外衣"，并由此成为席勒人物画"残缺变形"的内在解码框架。

## 主要贡献

### 核心著作（顾衡 074）

- 1899 《梦的解析》Die Traumdeutung
- 1905 《性学三论》Drei Abhandlungen zur Sexualtheorie

### 核心命题：意识两层 + 人格三块（顾衡 074 转述）

- **意识两层**：意识 / 潜意识
- **人格三块**：
  - **超我**——社会习俗灌输 → 意识层
  - **本我**——动物性 / 性欲 / 力比多 → 潜意识层
  - **自我**——战场，为本能和超我的厮杀提供场地

### 核心命题：[[神经官能症 Neurosis]] 的发病机制（顾衡 074）

- 超我太强大 → 打败本我 → 性欲 / 力比多被淤积 → 转化为种种躯体症状
- 配套概念：童年创伤性体验、俄狄浦斯情结、造梦的四大机制、阳具羡慕（小女孩儿）、阉割恐惧（小男孩儿）
- 顾衡评："反正什么都往性上扯。那时候的知识分子，一聊起文化来满嘴各种器官的。是这么个气氛。"

### 顾衡的德国民族性框架引申（074）

- "德国人也是个妥妥的大刺猬"（参考 [[刺猬与狐狸 Hedgehog and the Fox]]）——和法国人一样喜欢"包打一切的大真理"
- **法国 vs 德国**：法国人更注重"大真理"的普世性，德国人更注重终极性
- **弗洛伊德发明的"大真理"是性 / 力比多 / 本能**
- "在德国人看来，德国人比法国人更容易得神经官能症——因为德意志民族的超我非常强大，总能战胜鄙俗的本我，然后力比多就淤积了嘛！"

### 075 增补：力比多 / 仇女厌女症 → 席勒裸体画

顾衡 075 把弗洛伊德的影响整理成 075 论证链：

1. **力比多理论**——把人的几乎一切动机归结为"性"——直接催生席勒大量裸体画的创作热情
2. **神经官能症"按症状分成五个类型"**——比起法国实证主义、生命哲学**更容易切图**——席勒甚至**亲自跑到精神病院观察病人**
3. **席勒之"为什么瘦长"**——重度焦虑的人多半不是个胖子——所以席勒笔下人物总是**瘦长的和紧张的**
4. **席勒裸体画 = 力比多理论 + 仇女厌女症的图像化**——顾衡 075 收束："具体由谁来赋予意义？是新康德主义旗帜下寻找德意志民族性的知识分子，**以及把什么都往性上扯的弗洛伊德**。**这恰恰是理解德国表现主义的重中之重**。"

席勒名言互文：

> 否认性的人才是真正的淫秽，因为他们以最下作的方式侮辱了生他们的父母。

### 在 wiki 内的其他出场（076 视角）

- 076：作为 [[艾尔玛 Alma Mahler]] 与 [[马勒 Gustav Mahler]] 的婚姻危机咨询师——马勒严重恋母情结让艾尔玛"受不了"，两口子曾去找弗洛伊德分析，但**也不顶事儿**。一笔带过，作为 20 世纪初维也纳文化圈艺术家、音乐家、心理学家紧密交织的坐标存在。

## 师承与影响

- **私交**：[[克里姆特 Gustav Klimt]]（同在维也纳）—— 这条私交把精神分析输送给了维也纳分离派
- **影响**（艺术领域）：
  - [[席勒 Egon Schiele]]——精神分析是他人物画"残缺变形"的内在解码（[[神经官能症 Neurosis]] 五分类）
  - 整个 [[表现主义 Expressionism]] 维也纳支系的思想底色
- **影响**（流派外）：[[超现实主义 Surrealism]]（*not from wiki*：达利 / 布勒东等借用潜意识 / 梦的解析框架）
