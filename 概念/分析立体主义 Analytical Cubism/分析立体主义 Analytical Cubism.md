## 一句话定义

[[立体主义 Cubism]] 的第一阶段（约 1909–1912），由 [[毕加索 Pablo Picasso]] 与 [[勃拉克 Georges Braque]] 共同发展——**把对象砸成几何碎片，再让它们错位、翘起、翻转地重新拼回去**，试图同时呈现"多维空间 / 事物内部本质"。

## 详细解释

### 思潮根源

顾衡（066）把分析立体主义定位为**毕加索对 20 世纪初物理学革命的"字面误读"**——和他几年前误读 [[塞尚 Paul Cézanne]] 同质：

- **1904 汤姆森发现电子** → 古希腊 [[德谟克利特 Democritus|德谟克利特]] 以来"原子不可分"的传统崩塌；汤姆森提出"葡萄干布丁"原子模型（电子像葡萄干嵌在布丁里）。
- **1905 爱因斯坦相对论** → 静态三维空间观被打破；牛顿建立的"岁月静好"世界分崩离析。
- 两件事一起激发知识分子热议"多维空间"——"五维、六维、87 维"。
- 毕加索通过数学家 [[普朗塞 Maurice Princet]] 听了一耳朵公式，**听不懂**（"0–9 都写不出五个的学渣"），但**敏锐地意识到要迎合这股思潮**——往原子里走、往多维空间走。

### 方法论比喻

毕加索的解决方案：**"把一面镜子打碎，再让碎片错位、翘起、翻转地重新拼回去"**——

- 这种重组**不是按原样拼图**，而是有的错位、有的翘起、有的翻转，在画布上投下变形的影子。
- 目的：同时呈现"多维空间 / 原子内部本质 / 牛顿物理学崩塌后的支离破碎"。
- 名字"分析"由此而来：**像汤姆森揭示原子内部结构一样，通过绘画分析事物内在的本质**。

### 形式特征

- **单色调色**：放弃色彩、只用褐色 / 土黄 / 银灰——为与 [[马蒂斯 Henri Matisse]] 的 [[野兽派 Fauvism]] "唱对台戏"。把变量集中在形体上。
- **几何切面拼接**：整个画面由几何形状（小三角、小圆圈）拼接，碎片之间错位 / 翘起 / 翻转。
- **保留具象锚点**：人物面部仍点缀颜色作为提醒；后期加入字母、单词、熟悉题材（女人、乐器、酒瓶）防止辨认完全失败。
- **不签名**：毕加索与勃拉克作品多不签名，连他们自己都分不清谁画了哪幅——"都是些小三角小圆圈，有啥区别呢？"

### 根本悖论（顾衡判词）

要"画出本质"就得砸碎表象——但砸碎了别人就认不出来：

- 沃拉尔可识别 → 本质没画出
- 费尔南德 = 伊娃 → 表象消解 → 但"本质"成了"都是碳氢氧氮"——意义在哪？
- 两条退路：①只画熟悉题材；②保留字母 / 符号锚点
- 走到尽头：**翠花 = 汤唯 → 为何不直接画骷髅头？**

**矛盾解决不了 → 死胡同 → 转向 [[综合立体主义 Synthetic Cubism]]**。

### 不能被提问的画派

顾衡的另一个判词：分析立体主义是**第一个不能被问"你为啥要这么画"的画派**——

- 以前任何流派都能用一两句话讲清追求（[[现实主义 Realism|现实主义]] 拒斥宏大叙事、[[印象派 Impressionism|印象派]] 追求眼睛所见、[[象征主义 Symbolism|象征主义]] 主观的客观化、[[野兽派 Fauvism|野兽派]] 色与形的和谐）。
- 毕加索两次被人问到，**两次都是掏手枪朝天放枪赶人**（一次酒吧、一次出租车顶被打洞）。
- 顾衡的总评："艺术家能不能通过作品表达观念？能。前提是要先把要表达的东西搞清楚——而这恰是毕加索做不到的。"

### 勃拉克的主导作用

- **理念层面 [[勃拉克 Georges Braque]] 为主**——"聊相对论、聊多维空间，矮子里拔大个儿，两小儿中勃拉克是比较高的那个"。
- 勃拉克原是 [[野兽派 Fauvism]] Le Havre 小组成员（与 [[杜菲 Raoul Dufy]] 同乡），但痴迷 [[塞尚 Paul Cézanne]]——野兽派时期已显塞尚色彩（《[[希欧达的风景 Landscape at La Ciotat]]》、《[[安特卫普近郊的风景 Landscape near Antwerp]]》）。
- 初见《[[亚威农少女 Les Demoiselles d'Avignon|亚威农少女]]》骂"换吃木屑石蜡"，旋即顿悟"这是一边喝汽油一边点火柴"——回头与毕加索共创立体主义。
- 多年后承认：当年与毕加索聊过的话题"永远不会再提起"。

### 同时代反对

- **利奥·斯坦因**（[[格特鲁德·斯坦因 Gertrude Stein]] 的哥哥）：荒唐到了极点。
- **[[马蒂斯 Henri Matisse]]**：对西方艺术传统最严重的侮辱和冒犯，发誓让毕加索"沉下去"；毕加索回敬要让马蒂斯"知道自己一钱不值"。
- **[[阿波利奈尔 Guillaume Apollinaire]]**（站台）：毕加索画画像外科医生做解剖。
