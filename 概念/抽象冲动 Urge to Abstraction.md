## 一句话定义

[[沃林格 Wilhelm Worringer]] 1911 年提出的核心概念——**德意志—北方人的 [[艺术意志 Kunstwollen]]**，**与南方人 / 拉丁人写实主义相对立**，具体表现为 **[[向二维平面回归 Return to Flatness]]**（顾衡 076）。

## 详细解释

- **提出背景**：沃林格在 [[叔本华 Arthur Schopenhauer]] / [[李格尔 Alois Riegl]] / [[沃尔夫林 Heinrich Wölfflin]] 基础上，1911 年提出**地中海南方人与日尔曼北方人两种艺术形式的对比**。
  - **沃尔夫林**：比较了文艺复兴和巴洛克**两个不同时期**
  - **沃林格**：比较了欧洲南方人和北方人**两个种族 / 民族**——把维度从"时代"切到"民族"
- **沃林格的二元判断（顾衡 076 引）**：
  - **南方人**（古希腊→古罗马→意大利）：致力于**写实**——在二维画布上营造三维错觉
  - **北方人**（德国人）：像古埃及人、拜占庭人那样，倾向于绘画的**平面性**——这种平面性被沃林格定义为**一种抽象的冲动**
- **理论位置**：抽象冲动 ≠ 简单的"画得不像"，而是**民族艺术意志**的具体外化——是 [[表现主义 Expressionism]] 思想—画面建构的最终落脚点。
- **闭环（顾衡 076 小结）**：表现主义思想观念底层 = [[新康德主义 Neo-Kantianism]] → 把康德道德先验性以**民族 + 时代**两维窄化 → 让德意志民族性成为可想象之物 → 沃林格把这民族性的艺术表达定义为**抽象冲动** → 抽象冲动的具体表现 = [[向二维平面回归 Return to Flatness]] → **表现主义绘画理论建构完成**。
