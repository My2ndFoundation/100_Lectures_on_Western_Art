## 一句话定义

19 世纪末 [[象征主义 Symbolism]] 与 [[新印象主义 Neo-Impressionism]] **共享**的方法论核心——把抽象的主观（思想、情绪、观念）**编码**到客观世界的具体物象（颜色、形状、词语、画面）上，使其可被测量、计算，从而**纳入"科学"的合法性**。

由 [[古斯塔夫·康恩 Gustave Kahn]] 给出最简洁表述：

> "我们的主要任务是**主观的客观化**，而不是把客观主观化。"

## 详细解释

- **背景**：19 世纪后半叶 [[孔德 Auguste Comte]] [[实证主义 Positivism]] 影响下，"科学万能"成为西方思想气候。**美的规律 / 价值**是否能像物理化学一样**存在于外部客观世界**？这是这一时期诗与画都在追问的问题。
- **新印象主义路径**（[[昂里 Charles Henry]] + [[西涅克 Paul Signac]]《形感的教育》《色感的教育》）：
  - 颜色对比强度 → 中枢神经兴奋幅度 → 情绪反应（红黄快乐 / 绿紫蓝压抑）
  - 线条方向 → 情绪反应（左→右、下→上 快乐；右→左、上→下 压抑）
  - 昂里制作**和谐色搭配表格**
  - 自信"已为人类审美提供了具有普遍性的公式"
- **核心矛盾的克服**：人的感受发自内心（**主观**），如何让它**科学可测**？办法 = **把思想、情绪转化为具体的颜色 / 形状 / 字符串** → 可测量、可计算 → "科学"。
- **象征主义路径**（[[让·莫亚雷斯 Jean Moréas]] 宣言 + [[马拉美 Stéphane Mallarmé]] 诗学）：
  - 抽象观念 → 在客观世界中**寻找对应物**（爱情 ↔ 玫瑰、纯洁 ↔ 百合）
  - 对应物本身**退缩到阴影里去**——画 / 诗的真正主语是初发的思想
  - 马拉美工作流类比 [[斯金纳 B. F. Skinner]] **黑箱**：给输入（排版、标点、韵律、节奏） → 记输出 → 找函数关系
- **顾衡 048 金句**："**象征主义与新印象主义背后的理论支撑是一个东西，就是科学，就是'主观的客观化'。**"
- **"朝向"的关键性**：日常理解的象征是"具象 → 抽象"（用玫瑰表达爱情）—— [[加缪 Albert Camus]] 也是这样说的；但象征主义本身是"**抽象 → 具象**"——这个**朝向**的颠倒，是顾衡 048 反复强调的关键。
