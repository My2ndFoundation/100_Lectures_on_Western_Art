## 一句话定义

德语艺术史学的关键术语——把 [[叔本华 Arthur Schopenhauer]] 的"生命意志"借用到艺术领域，主张**艺术风格的发展有其内在的规律**、不受外界环境影响，本质乃**一种纯粹的形式**（顾衡 076）。

## 详细解释

- **概念脉络**：
  - **源头**：康德起头的"意志自由" → [[叔本华 Arthur Schopenhauer]] 完成"意志至上"的理论化和体系化（《作为意志和表象的世界》：世界的本质是意志，现象是意志的客体化）。
  - **艺术史学者的借用**：[[李格尔 Alois Riegl]] / [[沃尔夫林 Heinrich Wölfflin]] 都认为艺术风格的发展有其内在的规律——把"生命意志"借用为"艺术意志"。
  - **沃尔夫林石头滚下山坡的著名比喻**（顾衡 076 引）：
    > 艺术风格的演变就像一块石头滚下山坡，一路坑坑洼洼的不平，石头弹跳的轨迹是不规则的。但是，这块石头受了万有引力的影响，会一直向下。这个，就是艺术意志。
    - 石头一路遇到的坑坑洼洼 = 艺术与外界环境的互动（受外界影响）
    - 石头一直向下 = 艺术意志（**不**受外界影响，本质是**纯粹的形式**）
- **沃林格的扩展**：[[沃林格 Wilhelm Worringer]] 1911 在前辈基础上，把"艺术意志"具体化为**民族艺术意志**——明确提出**德意志民族艺术意志**这一概念，并把它定义为与南方意大利人、法国人**写实主义相对立的、一种 [[抽象冲动 Urge to Abstraction]]**；具体表现为 [[向二维平面回归 Return to Flatness]]。
- **重要不变项**：李格尔、沃尔夫林、沃林格三家都认同——艺术意志是**不受外界影响**的，是**纯粹形式**的内在驱动。
- **与 Willing 的关系**：艺术意志 = 叔本华 Willing 在艺术领域的投影；与个人 Emotion / Passion **相反**——含**不可抗拒的冷酷** + **对个人哭哭啼啼情感的蔑视**。

## 相关概念 / 流派 / 人物 / 技法

- 相关概念：[[抽象冲动 Urge to Abstraction]]、[[向二维平面回归 Return to Flatness]]、[[新康德主义 Neo-Kantianism]]、[[不屈的意志 Will to Power]]、[[移情说 Theory of Empathy]]（**对立**概念）
- 相关人物：[[叔本华 Arthur Schopenhauer]]（哲学源头）、[[李格尔 Alois Riegl]] / [[沃尔夫林 Heinrich Wölfflin]] / [[沃林格 Wilhelm Worringer]]（艺术史学借用者）
- 相关流派：[[表现主义 Expressionism]]（作为德意志民族艺术意志的画面对应物）
